This is my third time I've celebrated new years with my site, I even timed Human by Flavor Foley to new years with my clock on the index page! Well I was one second off, but eh close enough.

Every time the year number increments by one, I have a new task that is ready a new page for 2026 for blog and art sections. Well, I technically add a new page for blogs every quarter, but the process for a new year update is slightly longer.

Blog

Before I used 11ty, the blogs are in the form of collapsibles instead of indivual pages. The blogs of the current quarter are written on the index page, when a new quarter comes, I just move them to a new page.

  1. Add a new page (eg 2024a.html, 2024b.html)
  2. Update the sidebar and navbar blog selector to include the new quarter under the current year collapsible

But after I got 11ty, things got a bit more automatic. I don't have to add a new page for each quarter manually. However, I have other things to update now.

  1. Modify the index templates to show blogs in the current quarter by default
  2. Modify my template to use the current quarter tag
  3. Add the new page to the blog map page

Art

For my art page, they don't need to be updated quarterly as they only have one page per year. So the updating routine is just:

  • Add a new page for the year
  • Add a link to the new page in the art index page
  • Update the index page to show the newest art in the current year

For both instances, one thing I would really want to do is to add a new blog and art. It would feel somewhat empty if I don't add anything to the new pages, it would result in an awkward empyt page. So today, I've added a new art, and this new blog to celebrate new years... So yes this is blog is technically a filler :P
