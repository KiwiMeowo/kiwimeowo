With so many stuff done in this website, I'm running out of ideas on what to do with my site. Then, I had an idea to try out some obscure/less commonly used HTML tags. I remember someone else had this idea as well, though I have forgotten who.

<kbd>

When I looked at the blog of the aforementioned person, this is the tag I was interested the most. It is a tag for representing keyboard keys.

This is how it's used <kbd>Ctrl</kbd> + <kbd>V</kbd>.

This is how it's used Ctrl + V.

It is such a cute tag, I think I would want to use it in the future. I would style it like this: Ctrl + V.

<optgroup>

This is a tag designed to be used with the <select> tag. When you open the drop down menu, the items in the <optgroup> tag will be grouped into one:

  <select>
        <optgroup label="Group1">
            <option value="value1">Value1</option>
            <option value="value2">Value2</option>
        </optgroup>
        <optgroup label="Group2">
            <option value="value3">Value3</option>
            <option value="value4">Value4</option>
        </optgroup>
    </select>

There is no need to use the select menu yet, so I have never used this tag at the moment. But at the mean time, I should style it a bit:

<meter> and <progress>

Both of them shows a bar, but <meter> is for a level, while progress is for time stuff.

  <meter min="0" max="100" low="30" high="60" optimum="80" value="20">20%</meter>
  <br>
<meter min="0" max="100" low="30" high="60" optimum="80" value="50">50%</meter>
<br>
<meter min="0" max="100" low="30" high="60" optimum="80" value="90">90%</meter>
<br>
<progress max="100" value="70"> 70% </progress>
20%
50%
90%
70%

What's cool about the meter tag is that the color changes depending on the percentage. If it is lower than the low range, the bar will color red. If it is above the low range but lower than the optimum level, it will be yellow. Or else, it will be green.

The progress bar is very cool as well, but I have designed one that shows the number inside the bar, which is cooler IMO.

<dialog>

This acts like a popup window, and you can close the thing using a button encased in <form> tag:

  <dialog open="">
  <p>Ballkeeper</p>
  <form method="dialog">
    <button>OK</button>
  </form>
</dialog>
