Art RSS idea

This is an idea I have had in my mind for a while, I have an blog RSS, why not an art RSS? However, it initially sounds too annoying, especially if you do it manually. In an RSS feed, for every item, you have to put these elements (or at least how I format them):

  <item>
<title>[Title name]</title>
<link href="[Link]">[Link]</link>
<guid>[Some name]</guid>
<description>[Description]</description>
<pubDate>[Date]</pubDate>
</item>

The reason I have blog RSS set up is because I can have it automated using 11ty, I don't have to copy and paste that code and paste the details everytime I have a new blog.

To demonstrate how frustrated I would be if I have an art RSS, this is the normal procedure for uploading a new art

  1. Take a picture of my art
  2. Crop image and set filters
  3. Upload to my laptop
  4. Compress it
  5. Put it in my Dropbox folder
  6. Open my art page and copy/paste my art container
  7. Copy the link in Dropbox and paste it
  8. Set the thumbnail by adding style (though live preview window)
  9. Upload!

If I have to update the RSS manually, I have to do this everytime I upload a new art:

  1. Take a picture of my art
  2. Crop image and set filters
  3. Upload to my laptop
  4. Compress it
  5. Put it in my Dropbox folder
  6. Open my art page and copy/paste my art container
  7. Copy the link in Dropbox and paste it
  8. Set the thumbnail by adding style (though live preview window)
  9. Look for feed page (It's not easy finding individual files among lots of files)
  10. copy/paste item template
    1. Copy file name as title and paste
    2. Copy file link as link and guid, and paste
    3. Copy comments as description and paste
    4. Copy date as date and paste
  11. Upload!

Adding five more steps actually increases my mental workload by a lot, and going back and forth between files is very mentally draining. This is why I don't have an art ASS feed for a long time. However, I still wish to add an art RSS feed, since it would be easier for people to catch up with my updates.

Art RSS ideas?

Initally, I thought of my 11ty plugins. By using a json file containing my art in the _data folder, I can make a page generating the art elements and a RSS feed. It sounded good at first, but slowly I felt like the work won't be worth it:

  • The knowledge for 11ty is a bit specific, there isn't that much resources for it
  • The way I organise my art is by going through for loops in the amount of art divided by 18, then more for loops... It got too complex and I don't know how to do it in 11ty.

Moreover, it actually don't reduce my workload that well:

  1. Take a picture of my art
  2. Crop image and set filters
  3. Upload to my laptop
  4. Compress it
  5. Put it in my Dropbox folder
  6. Open _data/art.json at the very bottom of my site folder
  7. Copy/paste my art container
  8. Copy the link in Dropbox and paste it
  9. Run "npm run build"
  10. Go back up the my art page and open live preview
  11. Set the thumbnail by adding style (though live preview window)
  12. Run "npm run build" again
  13. Upload!

It's four more steps, two of them are running a terminal command. While npm run build is good at creating blog pages, it just feels tedious if it's for art. Also, I have to look through my folder too much for this solution, not good :/.

Solution

After all that 11ty code tinkering, I realised I have been overthinking the whole time. Just like how I implemented thumbnail.js for img styling, I can just add a code to copy an RSS item!

  1. Take a picture of my art
  2. Crop image and set filters
  3. Upload to my laptop
  4. Compress it
  5. Put it in my Dropbox folder
  6. Open my art page and copy/paste my art container
  7. Copy the link in Dropbox and paste it
  8. Set the thumbnail by adding style (though live preview window)
  9. Copy RSS item code (though live preview window as well!)
  10. Open and paste the RSS code into the feed
  11. Upload!

It's only two steps, and not much folder navigation needed, not much file writing needed either. It's just one copy, one paste, done! Man I'm so grateful I learnt Javascript.

I have added some optimization (like pinning the art gallery and the feed) so that I can make the process a bit easier. Anyways, if you want to follow my art gallery, now it's the perfect chance to add the feed to your RSS reader! If you don't have one, I believe Feedly (which I downloaded on my phone) works the best. It has no ads and simple to use, but there might be another good RSS feed readers out there!
